## 🔑 Chiavi Narrative — Indizi per il DM

### Force Grey
- **M1 → M2 → M3 → M4:** Cascata di eventi culmina in Nihiloor
- **M1:** Midnight Tears veleno = Cassalanters villain hint
- **M2:** Teschio d'oro = cruciale per M4 boss fight
- **M3:** Meloon posseduto = Divoratore di Intelletto rivelazione
- **M4:** Nihiloor = Mind Flayer orchestrating all

### Arpisti
- **M1 → M2 → M3 → M4:** Crescente scala minaccia Xanathar
- **M1:** Maxeene cavalla introduce informatore network
- **M2:** Inganno Xanathar = paranoia beholder
- **M3:** Doppelganger recruitment = Arpisti expand network
- **M4:** Infiltrazione Eyecatcher = Jarlaxle/Luskan rivelazione

### Zentharim
- **M1 → M2 → M3 → M4:** Intrigo fazioni & tradimento
- **M1:** Assassino drow = external threat
- **M2:** Skeemo traditore = Manshoon connection
- **M3:** Dasher's bite = soprannaturale element
- **M4:** Skeemo in fuga = Manshoon reveal, Torri di Kolat visible

## 🎲 Tiri Rapidi — Tabelle Omuncoli (Usa 1d20+ mod)

### Percezione (Check passivo normale = 10 + mod)

```
DC 10: Evidente, chiaro
DC 12: Nascosto, ma attento potrebbe notare
DC 13: Ben celato, attentivo specifico
DC 14: Molto nascosto, ricerca attiva
DC 15: Estremamente nascosto, ricerca professonale
DC 16+: Quasi impossibile senza aiuto magico
```

### Persuasione (Charisma base)

```
DC 10: PNG predisposto/amichevole
DC 12: PNG neutrale, ha tempo
DC 13: PNG indifferente, richiede sforzo
DC 14: PNG scettico, resistente
DC 15: PNG ostile, richiede grande sforzo
DC 16+: PNG determinato opposto, quasi impossibile
```
